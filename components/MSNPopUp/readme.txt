Instructions
============
  TMSNPopUp is tested with Delphi 4, 5 and 6. Please report it when you have another
  version of Delphi that works with the TMSNPopup component.

  Installation:
    - Start Delphi
    - Click the menu Component, Install Component
    - Point to the MSNPopUp.pas file
    - Click OK and recompile your user package

Use
===
  All properties are available through the Object Inspector, the component has
  only one event, which is fired when the user clicks somewhere in the popup
  window.
